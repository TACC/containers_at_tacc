|workshop_name| Workshop
==========================

|docs|

Course material at readthedocslink_

Software containers are an important common currency for portable and reproducible computing.  Learn best practices on building, using, and sharing Docker and Singularity containers in this hands-on workshop.  Also learn how to run those containers on |site| HPC systems, including MPI and GPU aware containers.

Topics will include:

- Docker and Singularity basics
- Containerizing your own code
- Running containers at |site|, including MPI parallelism and GPU enabled containers
- Integration with the module system
